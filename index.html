IQ Protocol JS SDK

IQ Protocol JS SDK Monorepo
:exclamation: The SDK is in development and breaking changes should be expected. Use at your own risk!

IQ Protocol JS SDK is a set of JavaScript packages which allows building applications powered by IQ Protocol. It includes libraries for communicating with IQ Protocol smart contracts and provides necessary tools for managing account state off-chain, using pluggable storage providers.
The SDK is designed with extensibility in mind, it indented to be blockchain and storage agnostic. It includes EVM-compatible blockchain provider and storage provider for Postgre SQL.
